# Game of Thrones BI Dashboard

Desktop-приложение для анализа данных о сражениях из вселенной Game of Thrones.

## Возможности

- Загрузка CSV-файлов
- Фильтрация данных
- Поиск по названию битвы
- KPI-статистика
- График по регионам
- График по годам

## Используемые технологии

- Python
- Pandas
- Matplotlib
- CustomTkinter
